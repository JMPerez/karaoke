Karaoke using Web Audio API

Do you like Karaoke? Find a music file and drag it here. If you are lucky, you will listen the song without vocals.

Drop your MP3 file here (also .ogg or .wav)

If you don't have any file at hand, click here for a demo of Happy by MMO.

How it works

By using the Web Audio API, the file is processed and voice is removed. Check out the project on Github for more information. No Flash was used to develop this.
